Multi Tier Architecture

What is a Multi Tier Architecture?

A software design where an application is divided into layers (tiers), such as:

    1. Presentation layer (UI)
    2. Application layer (logic)
    3. Data layer (database)
    4. This improves scalability and maintainability.
